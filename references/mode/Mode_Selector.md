# 模式选择器 (Mode Selector)

根据市场环境自动选择最适合的操作模式，指导当日交易决策。

---

## 核心功能

| 编号 | 功能 | 说明 |
|------|------|------|
| 1 | 判市 | 判断环境安全，能否开仓 |
| 2 | 选股 | 从 5000 只筛选重点关注池 |
| 3 | 诊股 | 单只股票五维打分 |
| 4 | 策略 | 制定买卖计划 |
| 5 | 复盘 | 每日总结与明日计划 |

---

## 工作目录

所有操作在当日工作区进行：

```
a-stock-YYYYMMDD/
├── 00-大盘上下文.md         # 判市结果
├── 01-策略大纲.md           # 选股结果
├── 02-标的观察池.md         # 观察池
├── trading-plan.json       # 交易计划（策略）
├── data/                   # 数据目录
│   ├── market_YYYYMMDD.csv
│   ├── sentiment_YYYYMMDD.json
│   ├── fund_flow_YYYYMMDD.json
│   ├── sector_rank_YYYYMMDD.json
│   ├── news_YYYYMMDD.json
│   └── tgb_YYYYMMDD_corpus.txt
├── logs/                   # 日志
└── 标的分析/              # 诊股结果
```

---

## 执行流程

```
┌─────────────────────────────────────────────────────────┐
│  Step 1: 判市                                             │
│  判断市场环境是否安全，决定仓位                           │
│  输出：00-大盘上下文.md + data/sentiment_YYYYMMDD.json   │
└─────────────────────────────────────────────────────────┘
                            ↓
┌─────────────────────────────────────────────────────────┐
│  Step 2: 选股                                             │
│  从全市场筛选重点关注池                                   │
│  输出：01-策略大纲.md + 02-标的观察池.md                  │
└─────────────────────────────────────────────────────────┘
                            ↓
┌─────────────────────────────────────────────────────────┐
│  Step 3: 诊股                                             │
│  对候选股票进行五维打分                                   │
│  输出：标的分析/XXX-股票名.md                            │
└─────────────────────────────────────────────────────────┘
                            ↓
┌─────────────────────────────────────────────────────────┐
│  Step 4: 策略                                             │
│  制定交易计划（买点/卖点/仓位/止损）                      │
│  输出：trading-plan.json                                 │
└─────────────────────────────────────────────────────────┘
                            ↓
┌─────────────────────────────────────────────────────────┐
│  Step 5: 复盘                                             │
│  每日收盘后总结                                           │
│  输出：更新 00-大盘上下文.md（明日展望）                   │
└─────────────────────────────────────────────────────────┘
```

## Step 1: 判市

### 数据来源

```python
# 获取市场情绪
ak.stock_zh_a_spot_em()                    # 全市场行情
ak.stock_main_fund_flow()                 # 主力资金
ak.stock_hsgt_fund_flow_summary_em()       # 北向资金
```

### 安全等级

| 等级 | 条件 | 建议仓位 |
|------|------|----------|
| 🟢 绿 | 涨停>50家，跌停<10家，指数上升 | 50-70% |
| 🟡 黄 | 涨停30-50家，跌停10-20家 | 20-30% |
| 🔴 红 | 涨停<30家，跌停>20家 | 0% |

### 输出文件

```markdown
# 大盘上下文 - 20251229

## 指数状态
- 上证：3265（+0.52%）
- 深证：10520（+0.63%）
- 创业板：2050（+0.78%）

## 市场情绪
- 涨停：52家 / 跌停：6家
- 情绪等级：🟢高涨

## 资金动向
- 北向：+48亿
- 主力：+62亿
- 热点板块：液冷服务器、储能、白酒

## 重要新闻
1. [政策] 新能源汽车补贴政策延续
2. [行业] 液冷服务器概念爆发
3. [公司] 圣阳股份7连板

## 安全等级：🟢 绿
## 建议仓位：60%
```

## Step 2: 选股

### 筛选条件

```
基础条件（必须满足）：
├── 非ST股
├── 流通市值 > 10亿
├── PE > 0
└── 上市时间 > 30天

风格条件：
├── 激进型：今日涨停、连板、热门板块
├── 稳健型：PE<20、MACD金叉、均线多头
└── 长线型：市值>100亿、ROE>15%、北向增持
```

### 输出文件

```markdown

2. 万科A（000002）- 低估值，技术转强
3. ...

1. 贵州茅台（600519）- 北向持续买入
```

## Step 3: 诊股

### 五维打分

| 维度 | 权重 | 指标 |
|------|------|------|
| 趋势 | 25% | 均线、MACD、KDJ |
| 资金 | 25% | 北向、主力、融资 |
| 人气 | 20% | 换手率、研报、热度 |
| 情绪 | 15% | 涨停次数、连板、溢价 |
| 基本面 | 15% | PE、PB、ROE、增速 |

### 评分等级

| 等级 | 分数 | 操作 |
|------|------|------|
| A | 85-100 | 重点买入 |
| B+ | 75-84 | 关注买入 |
| B | 65-74 | 观望 |
| C | 50-64 | 不推荐 |
| D | <50 | 排除 |

### 输出文件

```markdown
# 标的分析/000958-圣阳股份.md

## 综合评分：82/100 (B+级)

| 维度 | 得分 | 说明 |
|------|------|------|
| 趋势 | 85 | 均线多头，MACD金叉 |
| 资金 | 80 | 主力连续净买入 |
| 人气 | 72 | 换手率8%，讨论热度高 |
| 情绪 | 90 | 7连板，板块龙头 |
| 基本面 | 62 | PE偏高，基本面一般 |

## 结论
短期情绪驱动，可参与但需严格止损
```

## Step 4: 策略

### 策略要素

```json
{
  "code": "000958",
  "name": "圣阳股份",
  "entryPrice": 12.50,
  "stopLossPrice": 12.00,
  "takeProfitPrice": 14.40,
  "position": 0.20,
  "keyNode": "回踩MA5买入",
  "style": "波段"
}
```

### 仓位建议

| 评分 | 仓位 |
|------|------|
| A (85+) | 20-30% |
| B+ (75-84) | 15-20% |
| B (65-74) | 10-15% |

### 输出文件

`trading-plan.json` - 当日交易计划

## Step 5: 复盘

### 复盘内容

```markdown
# 大盘上下文 - 20251229（复盘更新）

## 今日操作总结
- 买入：圣阳股份，成本12.50
- 卖出：科陆电子，盈利+3.2%
- 持仓：圣阳股份，浮盈+8%

## 明日展望
- 情绪预期：延续（涨停52家，情绪高涨）
- 重点关注：圣阳股份（若回调可加仓）
- 风险提示：美股夜盘下跌，注意外盘影响
```

## 使用示例

```bash
# 完整流程执行
python scripts/mode_selector.py --step all

# 单步执行
python scripts/mode_selector.py --step market    # 判市
python scripts/mode_selector.py --step stock     # 选股
python scripts/mode_selector.py --step diagnosis # 诊股
python scripts/mode_selector.py --step strategy # 策略
python scripts/mode_selector.py --step review    # 复盘
```

## 状态流转

```
每日流程：
开盘前 → 判市 → 选股 → 诊股 → 策略 → 盘中监控 → 复盘 → 次日

状态标记：
- pending：待执行
- scanning：执行中
- done：已完成
- failed：失败
```
