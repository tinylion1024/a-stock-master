# 04 策略 (Trading Strategy)

为选中股票制定具体交易计划。

## 核心问题

```
什么时候买？买多少？怎么卖？
```

## 策略制定流程

### 4.1 买入时机（择时）

| 类型 | 信号 | 说明 |
|------|------|------|
| 打板 | 封板瞬间买入 | 追涨停，激进 |
| 回封 | 炸板后二次封板 | 确认强度 |
| 半路 | 回调至均线买入 | 折中方案 |
| 低吸 | 回踩支撑位买入 | 稳健方案 |
| 突破 | 放量突破新高 | 趋势确认 |

```
择时优先顺序：
1. 打板（最强信号，但风险高）
2. 回封（确认封板，较为安全）
3. 突破（趋势确认，稳健）
4. 半路（回调买入，均衡）
5. 低吸（支撑买入，最稳健）
```

### 4.2 仓位配置

```python
# 仓位计算公式
单票仓位 = 账户总资产 × 风险占比 ÷ 单股风险金额

# 示例
账户：100,000 元
风险占比：2%（单票最大亏损）
入场价：12.50
止损价：12.00（-4%）
每股风险：0.50 元

单票仓位 = 100000 × 0.02 / 0.50 = 4000 股
买入金额 = 4000 × 12.50 = 50,000 元
仓位占比 = 50%（重仓，需要控制）
```

| 评分等级 | 建议仓位 | 说明 |
|----------|----------|------|
| 🟢 A级（85+） | 20-30% | 优质可重仓 |
| 🟢 B+级（75-84） | 15-20% | 较好轻仓 |
| 🟡 B级（65-74） | 10-15% | 普通观望 |
| 🟠 C级（50-64） | 5-10% | 不推荐 |
| 🔴 D级（<50） | 0% | 排除 |

### 4.3 止损设计

```
止损原则：
1. 绝对止损：不抱幻想，触及必卖
2. 幅度控制：情绪 5%，量化 6%，机构 10%
3. 移动止损：浮盈后提高止损线
```

| 模式 | 固定止损 | 移动止损条件 |
|------|----------|--------------|
| 情绪（Fast） | -5% | 浮盈 > 10% → 提高到成本价 |
| 量化（Pro） | -6% | 浮盈 > 15% → 提高到 -3% |
| 机构（Ind） | -10% | 浮盈 > 20% → 提高到 +5% |

### 4.4 止盈设计

```python
# 分批止盈策略
if 涨幅 > 10%:
    卖出 30%（锁利）
if 涨幅 > 15%:
    卖出 30%（再锁利）
if 涨幅 > 20% 或 出现高位放量:
    卖出 40%（清仓）
```

| 模式 | 第一目标 | 第二目标 | 第三目标 |
|------|----------|----------|----------|
| 情绪（Fast） | +10% | +15% | +20% |
| 量化（Pro） | +12% | +18% | +25% |
| 机构（Ind） | +15% | +25% | +35% |

### 4.5 目标价计算

```python
# 基于技术面的目标价
def calculate_target_price(entry, pattern_type):
    if pattern_type == '突破':
        # 突破前高，目标涨幅 15-20%
        return entry * 1.18
    elif pattern_type == '连板':
        # 连板股，按板块平均高度估算
        return entry * 1.25
    elif pattern_type == '反弹':
        # 反弹目标 10-15%
        return entry * 1.12
    else:
        return entry * 1.15
```

## 关键规则

1. **买入不追涨超过 +3%**
2. **触及止损必须执行，不讨论**
3. **首仓不超过 20%，加仓不超 10%**
4. **单票最大亏损不超过总账户 2%**
