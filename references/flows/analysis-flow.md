# A股分析流程

## 分析流程概述

```
1. 市场环境判断 → 2. 板块选择 → 3. 个股筛选 → 4. 技术确认 → 5. 买入点决策
```

## 流程详解

### 第一阶段：市场环境判断

- 指数趋势：上证、深证、创业板指
- 情绪指标：涨跌停家数、成交量变化
- 资金流向：北向资金、主力净流入

### 第二阶段：板块选择

- 板块涨跌排行
- 板块资金净流入
- 板块内个股联动强度

### 第三阶段：个股筛选

- 基本面筛选：市值、PE、PB、ROE
- 技术面筛选：均线多头、突破关键价位
- 资金面筛选：主力持仓变化

### 第四阶段：技术确认

- K 线形态确认
- 指标共振检查
- 支撑压力位识别

### 第五阶段：买入点决策

- 分仓买入策略
- 止损止盈设置
- 持仓周期规划
