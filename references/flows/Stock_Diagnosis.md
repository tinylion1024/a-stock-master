# 03 诊股 (Stock Diagnosis)

对单只股票进行多维度质量打分。

## 核心问题

```
这只股票好不好？值不值得买？
```

## 五维诊断体系

```
诊股评分 = (趋势分 × 25%) + (资金分 × 25%) + (人气分 × 20%) + (情绪分 × 15%) + (基本面分 × 15%)
```

### 3.1 趋势诊断（25%）

| 指标 | 评估方法 | 得分 |
|------|----------|------|
| 均线系统 | MA5 > MA20 > MA60 → 多头 | 0-100 |
| MACD | DIF > DEA → 金叉 | 0-100 |
| KDJ | K > 50 → 强势区 | 0-100 |
| 趋势方向 | 上升/下降/震荡 | 0-100 |

```
趋势得分计算：
├── 均线多头排列：+30分
├── MACD 金叉：+25分
├── KDJ 强势区：+20分
├── 股价在 20 日均线上方：+15分
└── 创新高：+10分
满分：100分
```

### 3.2 资金诊断（25%）

| 指标 | 评估方法 | 得分 |
|------|----------|------|
| 北向资金 | 近5日净买入趋势 | 0-100 |
| 主力资金 | 近5日净流入趋势 | 0-100 |
| 融资融券 | 余额增加还是减少 | 0-100 |
| 成交额 | 量价配合是否健康 | 0-100 |

```
资金得分计算：
├── 北向净买入 > 1亿/日：+30分
├── 主力连续净流入 > 3天：+25分
├── 融资余额增加 > 10%：+20分
├── 成交额放大（量比 > 1.5）：+15分
└── 资金净流入为正：+10分
满分：100分
```

### 3.3 人气诊断（20%）

| 指标 | 评估方法 | 得分 |
|------|----------|------|
| 换手率 | > 5% 活跃，< 1% 低迷 | 0-100 |
| 人气排名 | 雪球/东财关注度 | 0-100 |
| 研报数量 | 近1月机构研报 | 0-100 |
| 讨论热度 | 社区讨论量变化 | 0-100 |

```
人气得分计算：
├── 换手率 > 10%：+30分
├── 人气排名 TOP 100：+25分
├── 有机构研报覆盖：+20分
├── 讨论热度上升：+15分
└── 成交额 > 5亿：+10分
满分：100分
```

### 3.4 情绪诊断（15%）

| 指标 | 评估方法 | 得分 |
|------|----------|------|
| 涨停次数 | 近5日涨停次数 | 0-100 |
| 连板情况 | 是否连板龙头 | 0-100 |
| 昨日表现 | 昨日涨停股今日高开 | 0-100 |
| 炸板率 | 封板失败率 | 0-100 |

```
情绪得分计算：
├── 近5日涨停 2次+：+30分
├── 连板中（2板以上）：+25分
├── 昨日涨停今高开 > 5%：+20分
├── 封板率高（炸板率 < 20%）：+15分
└── 板块情绪高涨：+10分
满分：100分
```

### 3.5 基本面诊断（15%）

| 指标 | 评估方法 | 得分 |
|------|----------|------|
| PE | 越低越好（< 20） | 0-100 |
| PB | 越低越好（< 3） | 0-100 |
| ROE | 越高越好（> 15%） | 0-100 |
| 净利润增速 | 越高越好（> 15%） | 0-100 |

```
基本面得分计算：
├── PE < 15：+30分
├── PE 15-30：+20分
├── ROE > 20%：+25分
├── ROE 10-20%：+15分
├── 净利润增速 > 30%：+25分
├── 净利润增速 15-30%：+15分
├── 市值 > 100亿：+15分
└── 行业龙头：+10分
满分：100分（但 PE 为负直接排除）
```

## 综合评分

```python
def calculate_diagnosis_score(stock):
    趋势分 = analyze_trend(stock) * 0.25
    资金分 = analyze_fund(stock) * 0.25
    人气分 = analyze_popularity(stock) * 0.20
    情绪分 = analyze_sentiment(stock) * 0.15
    基本面分 = analyze_fundamental(stock) * 0.15

    总分 = 趋势分 + 资金分 + 人气分 + 情绪分 + 基本面分
    return 总分
```

## 评分等级

| 等级 | 分数 | 说明 | 操作 |
|------|------|------|------|
| 🟢 A级 | 85-100 | 优质股票 | 重点关注，优先买入 |
| 🟢 B+级 | 75-84 | 较好股票 | 关注，可买入 |
| 🟡 B级 | 65-74 | 普通股票 | 观望，谨慎买入 |
| 🟠 C级 | 50-64 | 较差股票 | 不推荐买入 |
| 🔴 D级 | < 50 | 劣质股票 | 排除，不关注 |

## 关键规则

1. **总分 < 60 分 → 不关注**
2. **资金分为负 → 排除**
3. **基本面为负（PE<0）→ 排除**
4. **情绪分为0（无涨停）→ 短线不关注**
