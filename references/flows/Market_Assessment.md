# 01 判市 (Market Assessment)

判断当前市场环境是否适合开仓，综合行情、情绪、消息、资金、舆情五个维度。

## 核心问题

```
当前市场安不安全？能不能买股票？
```

---

## 一、指数状态（10%权重）

### 判断标准

| 指数 | 状态 | 判断条件 |
|------|------|----------|
| 上证 | 上升 | MA5 > MA20 > MA60，且价格在MA5上方 |
| | 震荡 | 均线缠绕，价格在均线间波动 |
| | 下降 | MA5 < MA20 < MA60，且价格在MA5下方 |
| 深证/创业板 | 同上 | 同上 |

### 指数权重

- 上证：40%
- 深证：35%
- 创业板：25%

---

## 二、情绪指标（30%权重）

### 2.1 涨停板情绪

| 涨停家数 | 等级 | 信号 |
|----------|------|------|
| > 80 | 极度亢奋 | ⚠️ 盛极而衰预警 |
| 50-80 | 高涨 | ✅ 安全 |
| 30-50 | 一般 | ⚠️ 谨慎 |
| < 30 | 低迷 | ❌ 危险 |

### 2.2 跌停板情绪

| 跌停家数 | 等级 | 信号 |
|----------|------|------|
| < 10 | 安全 | 无恐慌 |
| 10-20 | 谨慎 | 有分歧 |
| 20-40 | 恐慌 | ⚠️ 降仓 |
| > 40 | 极度恐慌 | ❌ 强制空仓 |

### 2.3 炸板率

```
炸板率 = 炸板数 / (涨停数 + 炸板数) × 100%

< 25% → 安全（封板成功率高）
25-40% → 一般
> 40% → 危险（封板意愿弱）
```

### 2.4 昨日涨停今表现

```
> 5% → 赚钱效应极强 ✅
3-5% → 赚钱效应强 ✅
0-3% → 赚钱效应一般 ⚠️
-3-0% → 亏钱效应 ⚠️
< -3% → 亏钱效应强 ❌
```

---

## 三、资金状态（25%权重）

### 3.1 北向资金

| 条件 | 信号 |
|------|------|
| 单日净买入 > 50亿 | 外资强势看多 |
| 单日净买入 30-50亿 | 外资看多 |
| 单日净买入 < 30亿 | 观望 |
| 单日净卖出 > 30亿 | 外资看空 |
| 连续净卖出 > 3天 | ⚠️ 谨慎 |

### 3.2 主力资金

| 条件 | 信号 |
|------|------|
| 单日净流入 > 100亿 | 主力强势做多 |
| 单日净流入 50-100亿 | 主力做多 |
| 单日净流入 < 50亿 | 观望 |
| 单日净流出 > 80亿 | 主力做空 |

### 3.3 融资余额

```
融资余额增加 > 5% → 杠杆资金入场，风险偏好提升
融资余额减少 > 5% → 杠杆资金离场，风险偏好下降
```

---

## 四、消息面（20%权重）

### 4.1 政策面

| 类型 | 信号 |
|------|------|
| 重磅利好（降准/政策支持） | 风险偏好提升 |
| 重磅利空（监管收紧） | 风险偏好下降 |
| 中性 | 无影响 |

### 4.2 行业消息

```
热点板块有重磅催化剂：
- 政策支持 → 板块持续性增强
- 无消息 → 热点难以持续
```

### 4.3 外盘影响

```
美股涨跌：
道指 ±0.5%以内 → 影响不大
道指 ±1%以外 → 注意明日开盘
纳指 > 1% 涨跌 → 科技板块受影响
```

### 4.4 消息面评分

| 维度 | 好（+） | 差（-） |
|------|--------|--------|
| 政策 | 降准/刺激政策 | 监管收紧 |
| 行业 | 政策支持/重大突破 | 行业利空 |
| 外盘 | 美股收涨 | 美股大跌 |
| 预期 | 乐观 | 悲观 |

---

## 五、舆情面（15%权重）

### 5.1 淘股吧情绪语料

从当日淘股吧热帖提取情绪关键词：

```python
# 情绪关键词
positive_keywords = ["涨停", "龙头", "确认", "主升", "满仓", "梭哈", "翻倍", "吃肉"]
negative_keywords = ["止损", "割肉", "空仓", "退潮", "亏钱", "被套", "踩雷", "股灾"]
neutral_keywords = ["复盘", "计划", "观察", "思考"]
```

### 5.2 舆情评分规则

```
高标反馈（连板股讨论热度）：

连板股讨论热度 > 1000 → 情绪高涨 ✅
连板股讨论热度 500-1000 → 情绪一般 ⚠️
连板股讨论热度 < 500 → 情绪低迷 ❌

热词占比：
正面词占比 > 60% → 情绪乐观 ✅
正面词占比 40-60% → 情绪中性 ⚠️
正面词占比 < 40% → 情绪悲观 ❌
```

### 5.3 高标反馈

关注最高标（市场最高连板股）的表现：

| 高标状态 | 信号 |
|----------|------|
| 高标继续涨停/连板 | 市场情绪高涨，风险偏好强 |
| 高标高位震荡未跌 | 情绪稳定 |
| 高标开始炸板/跌停 | 情绪退潮，警惕 |
| 高标跌停且无新标杆 | ❌ 情绪退潮，停止买入 |

---

## 六、综合评分

### 计算公式

```
总分 = (指数分 × 0.10) + (情绪分 × 0.30) + (资金分 × 0.25) + (消息分 × 0.20) + (舆情分 × 0.15)

满分：100分
```

### 评分等级

| 总分 | 安全等级 | 仓位建议 | 操作 |
|------|----------|----------|------|
| 80-100 | 🟢 绿 | 60-80% | 积极做多 |
| 60-79 | 🟡 黄 | 30-50% | 谨慎做多 |
| 40-59 | 🟠 橙 | 10-20% | 轻仓观望 |
| < 40 | 🔴 红 | 0% | 空仓等待 |

---

## 七、决策矩阵

### 七维决策表

| 情绪 | 资金 | 消息 | 舆情 | 高标 | 结论 |
|------|------|------|------|------|------|
| 高涨 | 流入 | 利好 | 乐观 | 连板 | ✅ 全力做多 |
| 高涨 | 流入 | 中性 | 乐观 | 连板 | ✅ 做多 |
| 一般 | 流入 | 中性 | 中性 | 震荡 | ⚠️ 谨慎做多 |
| 一般 | 流出 | 中性 | 中性 | 震荡 | ⚠️ 轻仓观望 |
| 低迷 | 流出 | 利空 | 悲观 | 跌停 | ❌ 空仓 |

---

## 八、风控红线（强制规则）

```
1. 跌停 > 30家 → 强制空仓（不看任何指标）
2. 高标跌停 > 3只 → 停止短线操作
3. 北向连续流出 > 5天 → 降仓至20%
4. 指数跌破20日均线 → 降仓至30%
5. 炸板率 > 50% → 停止打板
6. 舆情负面词占比 > 50% → 谨慎
```
