# 01 判市 (Market Assessment)

判断当前市场环境是否适合开仓。

## 核心问题

```
当前市场安不安全？能不能买股票？
```

## 分析维度

### 1.1 指数状态

| 指数 | 状态 | 标准 |
|------|------|------|
| 上证指数 | 上升/震荡/下降 | 均线多头/缠绕/空头 |
| 深证指数 | 同上 | 同上 |
| 创业板 | 同上 | 同上 |

### 1.2 情绪指标

| 指标 | 标准 | 含义 |
|------|------|------|
| 涨停家数 | > 50 → 安全 | 情绪高涨 |
| | 30-50 → 谨慎 | 中性 |
| | < 30 → 危险 | 情绪低迷 |
| 跌停家数 | < 10 → 安全 | 无恐慌 |
| | 10-20 → 谨慎 | 有分歧 |
| | > 20 → 危险 | 恐慌 |
| 炸板率 | < 30% → 安全 | 封板率高 |

### 1.3 资金状态

| 资金 | 指标 | 信号 |
|------|------|------|
| 北向 | 净买入 > 30亿 | 外资看多 |
| 主力 | 净买入 > 50亿 | 主力做多 |
| 融资 | 余额增加 | 杠杆资金入场 |

### 1.4 赚钱效应

```
昨日涨停股今日表现：
> 3% → 赚钱效应强，安全
0~3% → 赚钱效应一般
< 0% → 亏钱效应，危险
```

## 决策矩阵

| 涨停 | 跌停 | 指数 | 北向 | 结论 |
|------|------|------|------|------|
| >50 | <10 | 上升 | 流入 | ✅ 全力做多 |
| >50 | <10 | 震荡 | 流入 | ✅ 做多 |
| 30-50 | <10 | 上升 | 流出 | ⚠️ 谨慎做多 |
| 30-50 | 10-20 | 震荡 | 流出 | ⚠️ 轻仓观望 |
| <30 | >20 | 下降 | 流出 | ❌ 空仓 |

## 输出格式

```
【判市结果】

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
指数状态：上证 +0.5%（均线多头）
情绪指标：涨停 52家 / 跌停 6家 ✅
资金状态：北向 +48亿 ✅ / 主力 +62亿 ✅

综合评估：
├── 安全等级：🟢 绿色（可以开仓）
├── 市场情绪：高涨
├── 赚钱效应：强（昨日涨停 +4.2%）
└── 建议仓位：50%（轻仓试水）
    激进仓位：70%（情绪+资金共振）

风险提示：
├── 暂无系统性风险
└── 关注晚间美股表现
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

## 安全等级

| 等级 | 信号 | 仓位建议 |
|------|------|----------|
| 🟢 绿 | 可以做 | 50-70% |
| 🟡 黄 | 谨慎做 | 20-30% |
| 🔴 红 | 不能做 | 0% |

## 关键规则

1. **跌停 > 20 家 → 强制空仓**
2. **指数破位（跌破20日均线）→ 降仓至30%**
3. **北向连续流出 > 3 天 → 谨慎**
4. **炸板率 > 50% → 停止追板**
