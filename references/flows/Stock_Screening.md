# 02 选股 (Stock Screening)

从全市场 5000+ 只股票中筛选出重点关注池。

## 核心问题

```
哪些股票值得重点关注？
```

## 筛选流程

### 2.1 第一轮：基础筛选

```
条件（必须全部满足）：
├── 非 ST 股
├── 非 科创板（波动太大，新手不碰）
├── 上市时间 > 30 天
├── 流通市值 > 10 亿（流动性）
└── 市盈率 PE > 0（排除亏损股）
```

### 2.2 第二轮：量化筛选

| 类型 | 条件 | 说明 |
|------|------|------|
| 价值型 | PE < 20, PB < 3 | 低估值 |
| 成长型 | 净利润增速 > 15% | 业绩增长 |
| 技术型 | MACD 金叉 / 均线多头 | 技术信号 |
| 资金型 | 北向净买入 / 主力净流入 | 资金推动 |
| 突破型 | 突破20日新高 | 动量确认 |

### 2.3 第三轮：排序评分

```
综合评分 = (基本面分 × 30%) + (技术分 × 30%) + (资金分 × 25%) + (情绪分 × 15%)

各项满分 100 分：
├── 基本面分：PE、PB、ROE、市值
├── 技术分：均线、MACD、KDJ、RSI
├── 资金分：北向、主力、融资
└── 情绪分：涨停溢价、连板数、人气
```

### 2.4 第四轮：人工复核

```
复核要点：
├── 行业逻辑是否通顺
├── 消息面是否有催化剂
├── 股性是否活跃（换手率 > 3%）
└── 剔除庄股（分时不流畅）
```

## 关注池分类

### 激进型（适合 Fast 情绪模式）

| 条件 | 说明 |
|------|------|
| 今日涨停 | 情绪先锋 |
| 连板股 2-5 板 | 龙头确认 |
| 属于热门板块 | 板块助攻 |
| 流通市值 20-80 亿 | 便于炒作 |

### 稳健型（适合 Pro 量化模式）

| 条件 | 说明 |
|------|------|
| PE < 20 | 低估值 |
| MACD 金叉 | 技术转强 |
| 均线多头 | 趋势向上 |
| 主力净买入 | 资金认可 |

### 长线型（适合 Ind 机构模式）

| 条件 | 说明 |
|------|------|
| 市值 > 100 亿 | 大盘蓝筹 |
| ROE > 15% | 盈利能力 |
| 北向连续买入 | 外资背书 |
| 估值历史低位 | 安全边际 |

## 数量控制

```
关注池上限：
├── 激进型：≤ 10 只（聚焦龙头）
├── 稳健型：≤ 15 只（分散风险）
└── 长线型：≤ 8 只（重质不重量）

最终操作池：≤ 5 只（集中精力）
```

## 关键规则

1. **每日前 30 名涨跌幅榜必看**
2. **板块内涨停 > 3 只才纳入热点**
3. **ST 股永远不碰**
4. **上市不足 30 天不碰（次新股风险）**
