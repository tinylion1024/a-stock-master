# 05 复盘 (EOD Review)

每日收盘后进行市场总结和自我反思。

## 核心问题

```
今天操作怎么样？有什么需要改进的？
```

## 复盘框架

```
复盘 = 市场总结 + 操作检讨 + 明日计划
```

## 5.2 操作检讨

### 交易记录

```python
今日交易：
├── 买入：X 笔
│   ├── 标的：XXX，买入价 XX，理由 XX
│   ├── 标的：XXX，买入价 XX，理由 XX
│   └── ...
├── 卖出：X 笔
│   ├── 标的：XXX，卖出价 XX，盈亏 XX%
│   └── ...
└── 持仓：X 只
    ├── 标的：XXX，持仓价 XX，持仓盈亏 XX%
    └── ...
```

### 操作评分

| 检查项 | 评估 | 得分 |
|--------|------|------|
| 选股质量 | 选的是否符合模式 | /10 |
| 买点时机 | 买入时机是否最佳 | /10 |
| 仓位控制 | 仓位是否合理 | /10 |
| 止损执行 | 止损是否到位 | /10 |
| 止盈节奏 | 止盈是否合理 | /10 |

```
操作评分：
├── 选股：8/10（符合情绪模式）
├── 买点：7/10（回封板买入，时机较好）
├── 仓位：9/10（20%仓位，合理）
├── 止损：10/10（严格执行）
├── 止盈：6/10（卖早了，格局不够）
总分：40/50 (80%)
```

## 5.3 问题分析

```
问题清单：
1. 【买入】XXX 股票，买入时机偏早
   原因：追涨超过 +3%
   改进：等待回踩均线再买

2. 【卖出】XXX 股票，卖出时机不对
   原因：恐高心理导致卖早
   改进：设置移动止损，让利润奔跑

3. 【仓位】XXX 股票仓位过重
   原因：看好就重仓，忽视风控
   改进：单票仓位不超过 20%
```

## 复盘习惯

1. **每日必复盘**（收盘后 30 分钟内）
2. **亏损交易重点复盘**
3. **形成文字记录，积累经验**
4. **每周回顾，找规律**
