# A股大师 (A-Stock Master)

A 股智能投资分析助手，帮助投资者进行股票筛选、技术分析和投资决策。

## 功能特点

- 实时行情监控
- 技术指标分析
- 基本面筛选
- 板块热点追踪
- K线形态识别

## 目录结构

```
a-stock-master/
├── SKILL.md              # 技能定义文件
├── scripts/              # 脚本目录
├── references/           # 参考资料
│   ├── flows/            # 分析流程
│   └── guides/           # 分析准则
```
